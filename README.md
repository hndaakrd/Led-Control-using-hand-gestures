# Led-Control-using-hand-gestures
python code and arduino circuit for a project based on led controlled by hand gestures

first install the arudino code to the board by using ARDUINO IDE

to run the python code, use any IDE ( PYCHARM, VS CODE ).
install Pips if using VS code
PIP 1 : cv2
PIP 2 : mediapipe
PIP 3 : serial
PIP 4 : time
